# Getting Started

Get summon-claude running in under 10 minutes.

- **[Installation](https://summon-claude.github.io/summon-claude/latest/getting-started/installation/index.md)**

  Install summon-claude via uv, pipx, or Homebrew.

- **[Slack Setup](https://summon-claude.github.io/summon-claude/latest/getting-started/slack-setup/index.md)**

  Create and configure the Slack app for your workspace.

- **[Quick Start](https://summon-claude.github.io/summon-claude/latest/getting-started/quickstart/index.md)**

  Your first session from terminal to Slack.

- **[Configuration](https://summon-claude.github.io/summon-claude/latest/getting-started/configuration/index.md)**

  Run the setup wizard and verify your configuration.

- **[Set Up a Project](https://summon-claude.github.io/summon-claude/latest/guide/projects/index.md)**

  Create a project with a PM agent to coordinate multi-session workflows.

- **[Upgrading](https://summon-claude.github.io/summon-claude/latest/getting-started/upgrading/index.md)**

  Upgrade to the latest version.
