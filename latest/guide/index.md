# Guides

Learn how to use summon-claude effectively. Most users work through **projects** — a PM agent coordinates multiple Claude sessions on your behalf. You can also run quick ad-hoc sessions for one-off tasks.

- **[Projects](https://summon-claude.github.io/summon-claude/latest/guide/projects/index.md)**

  Multi-session project management. The primary way to organize work.

- **[PM Agents](https://summon-claude.github.io/summon-claude/latest/guide/pm-agents/index.md)**

  Autonomous project manager that spawns, directs, and monitors child sessions.

- **[Scribe](https://summon-claude.github.io/summon-claude/latest/guide/scribe/index.md)**

  Background monitoring agent for Slack and Google Workspace.

- **[GitHub Integration](https://summon-claude.github.io/summon-claude/latest/guide/github-integration/index.md)**

  PR review, code search, and GitHub tools.

- **[Scribe Integrations](https://summon-claude.github.io/summon-claude/latest/guide/scribe-integrations/index.md)**

  Google Workspace and Slack browser monitoring.

- **[Cron & Tasks](https://summon-claude.github.io/summon-claude/latest/guide/cron-tasks/index.md)**

  Scheduled jobs and task tracking.

- **[Canvas](https://summon-claude.github.io/summon-claude/latest/guide/canvas/index.md)**

  Persistent session documents.

- **[Lifecycle Hooks](https://summon-claude.github.io/summon-claude/latest/guide/hooks/index.md)**

  Run shell commands on worktree creation, project up/down.

- **[Configuration](https://summon-claude.github.io/summon-claude/latest/guide/configuration/index.md)**

  Customize summon-claude behavior.

- **[Sessions](https://summon-claude.github.io/summon-claude/latest/guide/sessions/index.md)**

  Session lifecycle, naming, and management.

Recommended reading order

New to summon-claude? Start with **[Projects](https://summon-claude.github.io/summon-claude/latest/guide/projects/index.md)** and **[PM Agents](https://summon-claude.github.io/summon-claude/latest/guide/pm-agents/index.md)**, then explore the features that interest you.
